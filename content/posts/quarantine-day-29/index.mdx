A modified Double Down from COMPTRAIN and a intense Core piece from Sonny Webster.

**METCON: "Double Down" for time**
* 100 [Line Hops](https://www.youtube.com/watch?v=4BaMG4OH7_g), 50 [Stand Ups](https://www.youtube.com/watch?v=Krd0XIU1SN0), 400m Run
* 80 Line Hops, 40 Stand-Ups, 400m Run
* 60 Line Hops, 30 Stand-Ups, 400m Run
* 40 Line Hops, 20 Stand-Ups, 400m Run
* 20 Line Hops, 10 Stand-Ups, 400m Run

*Notes*<br/>
If you have a Jump-Rope or a Speed-Rope, you can do Double Unders instead of Line Hops.<br/>
If you have a Bike or a Rower, you can use the machine instead of running.
